# CS-330-Comp-Graphic-and-Visualization
OpenGL 3D graphics project demonstrating transformations, texturing, lighting, and camera control by recreating real-world objects in a virtual scene.

This repository contains my work for CS 330 at Southern New Hampshire University. The course focuses on 3D graphics programming using OpenGL, including object modeling, transformations, texturing, lighting, and camera control.

For this course, these were used:
- C++
- OpenGL
- GLFW
- GLEW
- GLM
- GLSL (Shaders)
- Visual Studio

In this class, I transformed objects in the scene using translation, rotation, and scaling to create a realistic layout.
Textures were applied using UV coordinates and scaling to prevent stretching and improve visual quality.
Multiple light sources are used to simulate realistic lighting effects, including:
- Ambient lighting
- Diffuse lighting
- Specular highlights
We also had a movable camera allows the user to navigate and view the scene from different perspectives.
